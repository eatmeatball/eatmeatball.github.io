day09-MVC框架

1、mvc概念

2、psr规范

psr1：基础编程规范

psr2：编码风格规范

psr4：自动加载规范

5、命名空间映射

将命名空间和目录结构对应起来叫做命名空间映射

今天的代码命名空间的映射是一一对应的，大家可以自己改进代码，将一个命名空间对应多个目录？

'model' => 'app/model/'

'model\test' => 'app/model/lala'

'controller' => 'app/controller/'

'vendor' => 'vendor/lib'
